Expanse

Project 4 was a free-for-all. We were given the task of creating an app that illustrated the skills we feilt comfotable with. I chose to do another MERN stack with an emphasis on design and API consumption. Expanse is meant a simple wway to users to explore the anime medium with the information overload that can be found on the normal anime indexing sites. Using the Jikan API (MyAnimeList unofficial API), React, and Material UI, the end result was functional but not to the standard I was hoping. This is currently a work in progress. Users can create an account and login as well as search for specific titles.
